## 验证码
```
---------------- 请求 ----------------
{
    "api": "goback.login.captcha",
    "data": {
        "type":  "", // [string] 验证码类型，默认 login 
        "img_id": "", // [string] 验证码附加值
    }
}

---------------- 响应 ----------------
{
    "code": 10000,
    "msg": "",
    "data": {
        "img":   "", // [string] 图片base64编码
        "img_id": "", // [string] 验证码附加值，提交登陆的时候需要附带该参数
    }
}
```

## 登陆
```
---------------- 请求 ----------------
{
    "api": "goback.login",
    "data": {
        "type":     "", // [string] 登陆方式：password（密码登陆），phone（手机验证码），email（邮箱验证码）
        "username": "", // [string] 用户名
        "password": "", // [string] 密码，密码登陆方式的时候需要
        "code":     "", // [string] 验证码，如果没有开启验证码不需要
        "img_id":   "", // [string] 验证码附加值，如果没有开启验证码不需要
    }
}

---------------- 响应 ----------------
{
    "code": 10000,
    "msg": "",
    "data": {
        "session_id": "", //客户端身份标识session id
        "session_key": "", //签名key 
    }
}
```

## 注销
```
---------------- 请求 ----------------
{
    "api": "goback.logout",
}

---------------- 响应 ----------------
{
    "code": 10000,
    "msg": "",
}
```
