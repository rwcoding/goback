## 配置-列表
```
---------------- 请求 ----------------
{
    "api": "goback.config.list",
    "data": {
        "page":      1,  // [int] 分页
        "page_size": 1,  // [int] 每页显示几条数据 
    }
}

---------------- 响应 ----------------
{
    "code": 10000,
    "msg": "",
    "data": {
        "datas": [ // [array[object]] 列表数据
            {
                "id":   1,  // [int] 配置ID 
                "name": "", // [string] 名称
                "k":    "", // [string] 标识
                "v":    "", // [string] 数据  
            },
            ........
        ],
        "count":     1, // [int] 数据总数
        "page":      1, // [int] 当前页
        "page_size": 1, // [int] 每页显示条数 
    }
}
```
